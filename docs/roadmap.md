# Roadmap

- CLI `hermes agents`
- JSON output
- `agents.yaml` registry
- Handoff to Kanban/orch
- Terminal dashboard with Rich
- Plain text fallback
- Automated tests

3. Show tasks per agent.
4. Evolve to fullscreen TUI with Textual.
5. Turn into a plugin/extension, if Hermes supports that path cleanly.
