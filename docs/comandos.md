# Commands

## List agents

```bash
hermes agents
```

## List in JSON

```bash
hermes agents list --json
```

## Create initial registry

```bash
hermes agents init
```

## Handoff to orch

```bash
hermes agents orchestrate "my task"
```

## Open dashboard

```bash
hermes agents dashboard
```

## Dashboard in non-interactive mode

```bash
hermes agents dashboard --no-detail
```
